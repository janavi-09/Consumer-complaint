# Consumer Complaint Resolution

##  Project Overview
This project handles customer complaint text classification using machine learning. The dataset is cleaned, preprocessed, vectorized, and trained to classify complaints into appropriate categories.

##  Key Features
Train/Test dataset loading from Excel

Missing value handling and data type inspection

Text preprocessing and feature extraction

Machine learning model training + evaluation

##  Tech Stack
```
Python 路 Pandas 路 Scikit-Learn 路 NLTK 路 NumPy
```

##  Output
Predictive ML model for automated complaint classification.
